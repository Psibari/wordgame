# 🎨 1. BRAND IDENTITY

POLY WORDS is a **premium, modern vocabulary game** built around:

- Clarity  
- Confidence  
- Cognitive satisfaction  
- Playful intelligence  
- A mascot who roasts you just enough to keep you sharp  

Tone pillars:

- Smart  
- Witty  
- Encouraging  
- Slightly smug  
- Never childish  
- Never academic‑dry  

The brand feels like:

**NYT Games × Duolingo × a smug parrot who knows more words than you.**

# 🌈 2. COLOR PALETTE

Strict 3‑color rule (science‑backed for retention):

### **Primary Palette**
- **Deep Indigo** — `#1A1040`  
  Background, depth, premium feel  
- **Gold** — `#FFD700`  
  Accents, highlights, dopamine triggers  
- **Purple** — `#8B5CF6`  
  Creativity, glow, title letters  

### **Character Palette**
- **Polly Green** — mascot only  
- **Polly Orange Beak** — mascot only  

### Rules:
- No extra UI colors  
- No gradients except buttons  
- Dark background makes elements pop 4× more

# 🎮 5. GAME OVERVIEW

POLY WORDS is a **multi‑mode vocabulary mastery game** built around:

- Fast recall  
- Deep understanding  
- Pattern recognition  
- Memory retention  

The four modes form a **learning arc**:

1. **Blitz** — speed & instinct  
2. **Crossroads** — meaning & consequences  
3. **Scholar’s Cave** — depth & nuance  
4. **Echo Canyon** — memory & retention  

---

# ⚙️ 6. CORE SYSTEMS

### **Streak Set**
5 correct answers = 1 Streak Set  
Streak Sets power:

- XP multipliers  
- Polly reactions  
- Cosmetic unlocks  

### **Polly’s Discovery**
When a word is mastered:

- Added to Word Garden  
- Unlocks etymology  
- Unlocks fun facts  
- Unlocks usage examples  

### **CUT‑Safe Structure**
Every mode follows:

- **C** — Context  
- **U** — Understanding  
- **T** — Test

# 🕹️ 7. GAME MODES

---

## **BLITZ — Speed Mode**
**Tagline:** “Think fast. Faster.”  
**Goal:** Rapid meaning recognition  
**Structure:**  
- 10 words  
- 15 seconds each  
- 4 choices  
- Timer bar  
- Results screen  

---

## **CROSSROADS — Decision Mode**
**Tagline:** “Every word is a choice.”  
**Goal:** Apply meaning in context  
**Structure:**  
- Narrative scenarios  
- 2–3 choices  
- Branching outcomes  
- No timer  
- No decoys  

---

## **SCHOLAR’S CAVE — Depth Mode**
**Tagline:** “Go deeper.”  
**Goal:** Master nuance  
**Structure:**  
- Etymology  
- Synonyms  
- Antonyms  
- Usage  
- Mini‑tests  

---

## **ECHO CANYON — Memory Mode**
**Tagline:** “Say it again.”  
**Goal:** Long‑term retention  
**Structure:**  
- Spaced repetition  
- Recall prompts  
- Fill‑in‑the‑blank
